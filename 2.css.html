CSS 공부하기

html에 내가 css를 쓸거야 선언하기 : style 태그
style 안쓰고 속성으로 css 사용하기 : style 속성
style 태그 : 어디에 지정할지? => 선택자(selector) // a style 태그 : 무슨 효과를 설정? => 선언(declaration) // color:red;(property:value)

태그에 class로 그룹핑(html 기능), 띄어쓰기로 여러 class 부여 가능
클래스에 대해 스타일 지정 시 앞에 점(.) 붙여야함
태그에 id로 속성 추가(html에서 단 한번만 사용해야함) / 스타일 지정시 샾(#) 붙여야함
스타일 지정 선택자 우선 순위 : id(#) > class(.) > tag (동일 우선 순위 시 마지막에 등장하는거)
그 외 자세한 사항은 css selector 검색

박스모델 : 테두리 그리기 border로 태그가 차지하는 영역 확인
h1 tag = 한 줄 전체를 사용(block level element) / a tag = 해당 부분만 사용(inline element)
css에 display 속성을 추가해 바꿀수 있다.
f12에 style 탭에서 어떤 css style에 영향을 받는지 확인할 수 있다.
css 주석은 /* */
선택자 입력시 콤마(,)로 여러 선택자에 동일 옵션 적용 가능
div 태그 : 디자인을 위한 block element / span 태그 : 디자인을 위한 inline element
caniuse.com : 브라우저별로 해당 기능 사용 가능한지 / 얼마나 많은 사람이 쓰는지 통계를 볼 수 있는 유용한 사이트
