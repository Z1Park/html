JavaScript 공부하기

script 태그 : 여기부터 자바스크립트입니다를 나타내는 태그(html 내에서)
html은 정적인 설계(변경 불가) => 자바스크립트를 통해 동적으로 설계가능(계산도 가능)
documnet.write() 로 출력
버튼 추가하기 : input type='button' value='hi' onclick='alert('hi')'
onclick 속성 뒤에는 js가 와야함 / event 발생 시 js 문법에 따라 해석해서 웹브라우저가 동작
여러 event 종류로 반응형 웹 페이지를 만들 수 있다(검색 : javascript event)

크롬 : 개발자 도구에서 console을 통해 js 명령어 즉석 실행 가능
글자 수가 알고 싶다 : 복사 후 문자열 처리('') 후 .lenth
페이스북 댓글 중 임의로 몇 명 선택하기 => js를 안다면 간단하게 처리 가능

console 내에서 계산 명령 가능(산술 연산)
"문자열".property 로 여러 property 동작 가능(length, toUpperCase, indexof, trim 등)
문자열 + 문자열 로 concatnate 가능
console 내에서 변수 선언 가능(var)
